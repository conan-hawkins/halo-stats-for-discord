This code is for my own personal use.
This bot pulls information from the halo infinite API and uploads it to discord.

Commands prefix = #

#ranked = pulls all basic ranked stats.    <---->    example #ranked 'gamertag'

#stats  = pulls all basic halo stats, ranked and casual.    <---->    example #stats 'gamertag'

#help   = gives basic bot information.   <---->   example #help




Install versions.
Python 3.11.9       # Won't work with any version of python 3.12+ <-#inclusive
